# `laravel-parrot-setup`

A Parrot Project for laravel (vue) dev.

# Installation

```bash
composer require laravel-parrot/laravel-parrot-setup
```

* command for add parrot json with composer.json
```bash
laravel-parrot:setup
```
* composer dump-autoload
```bash
composer dump-autoload
```
